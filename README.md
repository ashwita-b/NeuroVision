**NeuroVision — MRI-Based Brain Tumor Analysis System**
An AI-powered web application that analyzes brain MRI scans to detect and classify brain tumors, with an integrated chatbot assistant.

**⚙️ Installation**
Step 1 — Clone or download the project
Step 2 — Create a virtual environment
Step 3 — Install dependencies 
      pip install -r requirements.txt

** 🚀 Running the System**
python app.py
Then open your browser and go to:
http://localhost:5000
